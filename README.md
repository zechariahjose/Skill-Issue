## Skill Issue

Skill Issue is a focused SQL and relational database learning website. It helps beginners build practical database skills through short lessons, quizzes, a SQL sandbox, and query challenges.

The site is centered on learning how databases work: tables, rows, columns, filtering, joins, aggregation, and writing queries that return useful results.

## Features

- SQL and relational database lessons
- Beginner-friendly explanations
- Interactive SQL sandbox with sample tables
- Quizzes with instant feedback
- Query challenges with automatic checking
- Progress tracking and study streaks

## Current Learning Path

- SQL & Relational Databases

## Running The Website

Open `index.html` in a browser. The project is a static site and does not require a build step.
